CSS Básico

Tipos de elementos

Existen dos tipos básicos de elementos en HTML:

Elementos de bloque
Son elementos que ocupan todo el ancho, empezando siempre en una linea nueva. Pueden contener márgenes. Ejemplos: p, div, ul, ol, li, dl, dt, dd, section, table, header, footer, hr, …

Elementos en linea
Son elementos que no empiezan en una nueva linea, solo ocupando el ancho que necesitan. Ejemplos: span, strong, em, input, select, textarea, …

Contenedores

main
Contenedor de bloque para el contenido
footer
Contenedor de bloque para el pie

Contenedor de bloque para la navegación
article
Contenedor de bloque para un artículo (muy recomendable para blogs)
aside
Contenedor de bloque para contenido adicional al contenido principal
section
Contenedor de bloque para una sección de la página

Genéricos

div
