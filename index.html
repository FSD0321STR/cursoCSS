Tipos de elementos

Existen dos tipos básicos de elementos en HTML:

Elementos de bloque
Son elementos que ocupan todo el ancho, empezando siempre en una linea nueva. Pueden contener márgenes. Ejemplos: p, div, ul, ol, li, dl, dt, dd, section, table, header, footer, hr, …

Elementos en linea
Son elementos que no empiezan en una nueva linea, solo ocupando el ancho que necesitan. Ejemplos: span, strong, em, input, select, textarea, …
