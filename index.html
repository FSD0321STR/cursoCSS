CSS Básico

Tipos de elementos

Existen dos tipos básicos de elementos en HTML:

Elementos de bloque
Son elementos que ocupan todo el ancho, empezando siempre en una linea nueva. Pueden contener márgenes. Ejemplos: p, div, ul, ol, li, dl, dt, dd, section, table, header, footer, hr, …

Elementos en linea
Son elementos que no empiezan en una nueva linea, solo ocupando el ancho que necesitan. Ejemplos: span, strong, em, input, select, textarea, …

Podemos forzar el cambio de comportamiento de un elemento mediante la propiedad display

                        display:block;
                        display:inline;

Margins, Borders, Paddings, Height, Width

Los elementos de bloque tienen unas propiedades relacionadas con el tamaño del elemento y el espaciado. Son:

Height, width

Marcan el alto y el ancho del elemento

Margins

Marcan el espacio que debe existir entre el borde del elemento y los elementos adyacentes.

Se especifica con las siguientes síntaxis

                        //notación específica
                        margin-top: 10px;
                        margin-right: 10px;
                        margin-bottom: 10px;
                        margin-left: 10px;

                        //notación simplificada general
                        margin: 10px;

                        //notación simplificada específica
                        margin: 10px 10px 10px 10px;

Border

Marcan el grosor y el estilo del borde del elemento

                        //notación específica
                        border-top: 10px solid red;
                        border-right: 10px solid red;
                        border-bottom: 10px solid red;
                        border-left: 10px solid red;

                        //notación simplificada general
                        border: 10px solid red;

                        //redondeos
                        border-radius: 5px;
                        border-top-left-radius: 5px;

Padding

Marcan el espacio que debe existir entre el borde del elemento y el propio elemento

                        //notación específica
                        padding-top: 10px;
                        padding-right: 10px;
                        padding-bottom: 10px;
                        padding-left: 10px;

                        //notación simplificada general
                        padding: 10px;

                        //notación simplificada específica
                        padding: 10px 10px 10px 10px;

Textos

CSS facilita estilar muchos aspectos de la tipografía

font-family

Establece la tipografia del elemento

Establece el tamaño de la fuente

                        font-size: 14px;
                        font-size: 2rem;

Establece el grosor de la fuente

                        font-weight: 800;

font-stretch

Establece el ancho de la fuente

                        font-stretch: condensed;
                        font-stretch: expanded;

text-align

Establece la alineación del texto dentro del elemento de bloque

                        text-align: center;
                        text-align: justify;

text-decoration

                        text-decoration: underline;
                        text-decoration: underline overline dotted red;

text-indent

Establece el espaciado de la primera linea de texto

                        text-indent: 5rem;

text-transform

Transforma el texto

                        text-transform: uppercase;
                        text-transform: lowercase;

line-height

Pseudo clases

Sirven para diferenciar estados de un element. Se marcan con el símbolo ":"

:link
Enlace no visitado
:visited
Enlace visitado
:hover
Mouse encima del elemento
:active
Enlace seleccionado
:focus
Elemento con focus
:first-child
Primer elemento hijo
:last-child
Último elemento hijo
:nth-child(n)
Elemento "n" hijo

Pseudo elementos

Sirven para estilar la primera linea o la primera letra de un elemento, o para añadir contenido antes o después del elemento. Se marcan con el símbolo "::"

                        p::first-letter {
                            color: #ff0000;
                            font-size: xx-large;
                            }
                        
                        p::first-line {
                            color: #0000ff;
                            font-variant: small-caps;
                            }
                        h1::before {
                            content: url(smiley.gif);
                            }
                        h1::after {
                            content: url(smiley.gif);
                            }

Tablas

Las tablas tablas pueden tener tres secciones que pueden servir para diferenciar estilos (thead, tbody, tfooter)

Además podemos marcar una celda como cabecera mediante th

Header - Celda 1 Header - Celda 2 Header - Celda 3
Fila 1 - Celda 1 Fila 1 - Celda 2 Fila 1 - Celda 3
Fila 2 - Celda 1 Fila 2 - Celda 2 Fila 2 - Celda 3
Footer - Celda 1 Footer - Celda 2 Footer - Celda 3

En realidad podemos convertir cualquier elemento en una table mediante la propiedad "display"

display: table
Convierte el elemento en un contenedor de tabla
display: table-row
Convierte el elemento en una fila de la tabla
display: table-cell
Convierte el elemento en una celda de la tabla
display: table-header-group
Convierte el elemento en un contenedor "thead"
display: table-row-group
Convierte el elemento en un contenedor "tbody"
display: table-footer-group
Convierte el elemento en un contenedor "tfoot"

Posiciones

Podemos modificar la posición de un elemento mediante la propiedad "position"

static
Valor por defecto, el objeto se situa en el orden que le pertoca
absolute
El elemento queda flotante y podemos situarlo donde deseemos.
fixed
El elemento quedará situado relativamente a la ventana del navegador (modales)
sticky
El elemento quedará situado basado en el scroll del usuario
                    position: static;
                    position: absolute;
                    position: fixed;
                    position: sticky;

CSS Avanzado

Media queries

Nos permiten definir diferentes declaraciones CSS según el tamaño o resolución de los dispositivos.

                        @media not|only mediatype and (expressions) {
                            Código CSS
                        }
                        
                        @media screen and (min-width: 480px) {
                            body {
                                background-color: lightgreen;
                            }
                        }

Variables

Permiten especificar variables en CSS como si estuvieramos programando

                        :root {
                            --primary-color: #1e90ff;
                            --secondary-color: #ffffff;
                        }

                        body {
                            background-color: var(--primary-color);
                        }

                        /* modificar variables en media queries */
                        .container {
                            --fontsize: 25px;
                        }

                        @media screen and (min-width: 450px) {
                            .container {
                              --fontsize: 50px;
                            }
                        }

Animaciones

Con CSS podemos crear animaciones de los elementos HTML. Para ello primero debemos de definir los keyframes de la animación

                        @keyframes example {
                            from {background-color: red;}
                            to {background-color: green;}
                        }

                        @keyframes example {
                            0%   {background-color: red;}
                            25%  {background-color: yellow;}
                            50%  {background-color: blue;}
                            100% {background-color: green;}
                          }

Y a continuación podemos aplicarlo a cualquier elemento mediante estas declaraciones:

                        div {
                            animation-name: example; /* ligado al indicador de la animación ("example") */
                            animation-duration: 4s; /* duración de la animación */
                          }

Las animaciones tienen una serie de propiedades que permiten controlarlas

                        animation-delay: 2s /* Esperar 2s para iniciar la animación */
                        animation-iteration-count: 3 /* Repetir la animación 3 veces (puede ser "infinite") */
                        animation-direction: reverse /* Permite controlar la dirección de la animación (normal, reverse, alternate, altermate-reverse) */
                        animation-timing-function: ease /* Permite controlar la aceleración de la animación (linear, ease, ease-in, ease-out, ease-in-out) */ 
                        animation-fill-mode: forwards /* Permite definir el último estado después de ejecutarse la animación (none, forwards, backwards, both) */
